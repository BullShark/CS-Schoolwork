COSC2415 Data Structures Spring 2006

Programming Assingment 2: The GUI Infix to Postfix Converter

Due on 3/22/2006

I. Program Descritpion: (100 points)

For this programming assignment, you will develop a Java GUI Infix to Postfix Converter that takes user input from the GUI keypad you develop, convert the infix to postfix expression using Stack data structure and the algorithm we discussed in class. Here is the converter user interface.

The converter will:

  1. support all *, /, +, -, (, ), and ^ conversion operation.
  2. have a "Clear" button to clear the input.
  3. have a "Convert" button to convert the infix expression to postfix.

II. Extra Credits (25 points)

Add a "Evaluate" button to evaluate a postfix expression using the algorithm we discussed in class.

III. Hand-In:

  1. A UML diagram of your design. It should be drawn using PowerPoint or equivalent.
  2. Your source codes listings.
  3. One page, single space report describe your project, and known bugs.
In addition to the hand-in, please zip the folder that contains your files for this assignment and e-mail it to me at yng@accd.edu.
